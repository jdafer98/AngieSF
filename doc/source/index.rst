Breve descripción.
^^^^^^^^^^^^^^^^^^

   El proyecto se concibe como una agrupación de herramientas ligeras de uso cotidiano en el ámbito de la ciberseguridad. El proyecto consiste en un conjunto de herramientas con interfaz cli escritas en python con el objetivo de agrupar funciones básicas necesarias en el terreno de la ciberseguridad y llevarlo a un entorno sencillo, ligero, elegante y portable.
